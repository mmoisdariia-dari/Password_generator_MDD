# Генератор случайных паролей с оценкой сложности

Простая программа на Python для генерации безопасных паролей, проверки их сложности и сохранения в зашифрованном виде.

## 📋 Описание

Проект выполняет следующие функции:
- Генерация случайных паролей с настраиваемыми параметрами
- Проверка сложности пароля (оценка от 0 до 100)
- Шифрование и сохранение паролей в файл
- Простой консольный интерфейс
- Автоматические тесты

## 🚀 Возможности

### Генерация пароля
- Настраиваемая длина (по умолчанию 12 символов)
- Выбор использования:
  - Заглавных букв (A-Z)
  - Строчных букв (a-z)
  - Цифр (0-9)
  - Специальных символов (!@#$%^&*)

### Проверка сложности
Программа оценивает пароль по двум критериям:
1. **Длина пароля** (до 40 баллов)
2. **Разнообразие символов** (до 60 баллов)

Уровни сложности:
- **80-100 баллов** — ОЧЕНЬ СЛОЖНЫЙ
- **60-79 баллов** — СЛОЖНЫЙ
- **40-59 баллов** — СРЕДНИЙ
- **0-39 баллов** — СЛАБЫЙ

### Шифрование
Пароли сохраняются в зашифрованном виде с использованием кодировки Base64.
