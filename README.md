# Генератор случайных паролей с оценкой сложности

Программа на Python для генерации безопасных паролей, проверки их сложности с использованием ИИ-анализа и сохранения в зашифрованном виде.

## 📋 Описание

Проект включает:
- ✅ Генерацию случайных паролей с настраиваемыми параметрами
- ✅ Проверку сложности пароля (оценка от 0 до 100 + 5-звёздочный рейтинг)
- ✅ ИИ-анализ пароля на основе готовых данных
- ✅ Шифрование и сохранение паролей в файл
- ✅ Простой консольный интерфейс
- ✅ Автоматические тесты

## 🚀 Возможности

### Генерация пароля
- Настраиваемая длина (по умолчанию 12 символов) и защита от некорректного ввода (принимаются только числа)
- Выбор использования:
  - Заглавных букв (A-Z)
  - Цифр (0-9)
  - Специальных символов (!@#$%^&*()-_=+)

### Проверка сложности

**Новая система оценки:**
- **Длина пароля** — до 65 баллов
- **Разнообразие символов** — до 35 баллов
- **Итого** — 100 баллов

**5-звёздочный рейтинг:**
- ⭐⭐⭐⭐⭐ (90-100 баллов) — ОЧЕНЬ СЛОЖНЫЙ
- ⭐⭐⭐⭐☆ (75-89 баллов) — СЛОЖНЫЙ
- ⭐⭐⭐☆☆ (60-74 балла) — СРЕДНИЙ
- ⭐⭐☆☆☆ (40-59 баллов) — СЛАБЫЙ
- ⭐☆☆☆☆ (0-39 баллов) — ОЧЕНЬ СЛАБЫЙ

**Важно:** пароли короче 8 символов всегда получают 1 звезду, независимо от баллов!

### ИИ-анализ пароля

Функция `ai_password_check()` анализирует пароль по множеству критериев:
- Проверка на наличие в списке популярных (слабых) паролей
- Поиск простых паттернов (123, abc, qwerty)
- Обнаружение клавиатурных паттернов (qwerty, asdf, qazwsx)
- Проверка на однотипность (только цифры или только буквы)
- Анализ повторяющихся символов
- Проверка на простые комбинации (цифры+буквы)
- Выдача рекомендаций по улучшению

### Шифрование
Пароли шифруются с использованием:
- Алгоритма **SHA-256** (криптографическое хеширование)
- **Секретной соли** для защиты от атак по словарю
- Кодировки **Base64** для удобного хранения

При проверке пароль сравнивается с сохранённым хешем. Это **одностороннее шифрование** — пароль нельзя расшифровать обратно, но можно проверить его корректность.

### Безопасность ввода
- Проверка пароля на наличие кириллицы (принимаются только латинские символы)
- Валидация длины пароля (только числовые значения)
